embroidery

inkscape embroidery vp3 output plugin based on [Jon Howell,2010] + standalone vp3 rw class.py

to install:

linux:
  copy files: 
    PyEmb.py 	
    embroider.inx 	
    embroider.py 	
    vp3rw.py
  into inkscape plugin directory: ~/.config/inkscape/extensions
